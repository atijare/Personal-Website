Featured Project Story

WiseWallet

I built WiseWallet to solve a simple problem: most budgeting tools are either too complex or too shallow. The goal was to keep the interface intuitive while still delivering serious features like secure auth, smart receipt parsing, and useful reporting.

  • Idea: Make budget tracking feel like a habit, not a chore.
  • Build: Started with auth + core expense flow before adding AI layers.
  • Learned: Users stay longer when setup friction is nearly zero.

Challenge: Balance security, usability, and AI-assisted automation in one coherent product.

Outcome: A full-stack app that helps users track spending habits with less manual effort.
